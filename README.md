# HKFITI

**HKFITI: A herb-knowledge–guided heterogeneous graph neural network for ingredient–target interaction prediction**

**Tested Environment:**  
- Python 3.12  
- PyTorch 2.2.0  
- CUDA 12.1  


### 1. Herb–Ingredient–Target Dataset  
Download link:  
[Google Drive Dataset](https://drive.google.com/drive/folders/1iCXN-0SsLPFX7Vt4_ve2La0JnZKf_vf5?usp=drive_link)  
After downloading, **extract the files into the `data/` directory**.

### 2. BindingDB & DrugBank Datasets  
These datasets can be obtained from:  
[MMDG-DTI GitHub Repository](https://github.com/JU-HuaY/MMDG-DTI)
